Extensão para CHROME para mostrar a resposta certa para o `curso.dev/show`

Como utilizar:

- Baixe esse repositório em uma pasta
- Abra o Chrome → chrome://extensions
- Ative Modo desenvolvedor
- Clique em "Carregar sem compactação"
- Selecione a pasta
